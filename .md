Bạn là một kỹ sư phần mềm senior và chuyên gia AI/ML. Hãy xây dựng một hệ thống hoàn chỉnh có tên:

"AI Test Analytics Dashboard for Software Quality Evaluation"

🎯 MỤC TIÊU:
Phát triển một dashboard tương tác giúp đo lường hiệu quả kiểm thử phần mềm bằng cách sử dụng các số liệu kiểm thử và phân tích có sự hỗ trợ của AI.

====================================
📥 INPUT DATA SOURCES:
Hệ thống phải tích hợp và xử lý dữ liệu từ:
1. Test Case Repository (file JSON/CSV hoặc database)
2. Test Execution Reports (log hoặc report)
3. Issue Tracking Systems (GitHub Issues hoặc JIRA API)
4. Code Repository (GitHub API để lấy metadata hoặc coverage nếu có)

====================================
⚙️ PROCESSING LOGIC:

1. TÍNH TOÁN TEST EFFECTIVENESS METRICS:
- Test Execution Rate = Executed Test Cases / Total Test Cases
- Pass Rate = Passed / Executed
- Fail Rate = Failed / Executed
- Defect Detection Percentage (DDP) = Defects found during testing / Total defects
- Defect Leakage Rate = Defects found after release / Total defects

2. TÍNH TOÁN TEST COVERAGE METRICS:
- Requirement Coverage = Covered Requirements / Total Requirements
- Code Coverage (nếu có tool như JaCoCo)
- Test Case Coverage = Test Cases mapped to features / Total features

3. PHÂN TÍCH XU HƯỚNG:
- Biểu đồ defect theo thời gian
- Biểu đồ pass/fail theo thời gian
- Trend coverage

====================================
🤖 AI COMPONENT (BẮT BUỘC):

Xây dựng module AI sử dụng Python (scikit-learn hoặc TensorFlow):

1. Risk Prediction:
- Input: historical test data
- Output: dự đoán module nào có nguy cơ lỗi cao
- Model gợi ý: RandomForest / Logistic Regression

2. Test Gap Detection:
- Phát hiện khu vực chưa được test đầy đủ (coverage thấp)

3. Test Case Recommendation:
- Gợi ý thêm test case dựa trên pattern lỗi trước đó

====================================
📊 OUTPUT DASHBOARD:

Xây dựng dashboard web với các thành phần:

Frontend:
- html,css

Backend:
- NodeJS (Express) hoặc Python (Flask/FastAPI)

Dashboard phải hiển thị:
- Test effectiveness metrics (card + chart)
- Coverage metrics (progress bar + chart)
- Defect trend (line chart)
- Risk heatmap (AI prediction)
- Suggested test cases (AI output)
